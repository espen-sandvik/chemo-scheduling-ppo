# Single-Drug Chemotherapy Scheduling Using PPO

Supplementary code for the master's thesis:

Espen Sandvik Havstein — Høgskolen i Molde, Logistics Analytics, 2026

## What this repository contains

| File / Folder | Description |
|---|---|
| `Baseline_final_3600.ipynb` | Main Jupyter notebook: Gymnasium environment (`HydroxyureaEnv`), PPO training, OAT sensitivity sweep, and result plots |
| `main.py` | Identical code exported as a plain Python script for readers without Jupyter available |
| `Results_figures/` | Output figures generated by the notebook, as they appear in the thesis |
| `requirements.txt` | Python package dependencies |


## How to run

### Option A — Jupyter notebook (recommended)

```bash
pip install -r requirements.txt
jupyter notebook Baseline_final_3600.ipynb
```

Run the cells in order. The notebook is self-contained: it defines the environment, trains the PPO agent, and generates all result plots.

### Option B — Plain Python script

```bash
pip install -r requirements.txt
python main.py
```

`main.py` is a direct export of the notebook.
